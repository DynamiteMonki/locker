# Functions
 
 A function is a named block of code that performs a specific task and can be called from other parts of a program to execute that task

---

# Defination of function

A *function is defined with a template* that if not followed the code doesn't work.

```text 
<exit-type> <function-name>(<input/s-types>) {body(what it does)}
```
